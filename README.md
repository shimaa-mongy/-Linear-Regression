# Student Score Prediction using Simple Linear Regression

## 📌 Project Description

This project demonstrates how to use **Simple Linear Regression** with **Scikit-learn** to predict student scores based on the number of hours they studied.
The model analyzes the relationship between **study hours (Hours)** and **student scores (Scores)** and then builds a predictive model.

## 🛠️ Technologies Used

* Python
* NumPy
* Pandas
* Matplotlib
* Seaborn
* Scikit-learn

## 📊 Project Steps

1. Import the required libraries for data analysis and modeling.
2. Load the dataset from a CSV file.
3. Explore the dataset using:

   * `info()`
   * `describe()`
4. Visualize the relationship between study hours and scores using **Seaborn**.
5. Split the dataset into:

   * Training data (70%)
   * Testing data (30%)
6. Build a **Linear Regression** model.
7. Train the model using the training dataset.
8. Predict student scores using the test dataset.
9. Evaluate the model performance using:

   * Mean Absolute Error (MAE)
   * Mean Squared Error (MSE)
   * R² Score

## 📈 Project Goal

The goal of this project is to understand the fundamentals of **Linear Regression** and how it can be used to predict numerical values based on historical data.
